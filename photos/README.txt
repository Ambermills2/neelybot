NeelyBot — Family Photos
========================

Drop your photo files right here in this "photos" folder
(JPG, PNG, WEBP, GIF, or HEIC).

Then list their file names in manifest.json, like this:

[
  "calf.jpg",
  "aspen-lake.jpg",
  "family-yosemite.jpg"
]

That's it. The Photos screen and the slideshow will use them automatically,
in the order listed. If manifest.json is empty ( [] ), the app just shows the
sample gradient art instead — nothing breaks.

Notes
- Keep file names simple: letters, numbers, dashes. No spaces if you can help it.
- The slideshow shows each photo for the number of seconds set on the Photos
  screen ("Each Photo"), and Shuffle mixes the order.
- PRIVACY: the hub is served from a public web address, so anything in this
  folder is reachable by URL once deployed. Ranch / animal / scenery shots are
  fine to publish now. Hold close-ups of the kids until the family login is in.
